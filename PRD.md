# PRD: ТопСпин — приложение для настольного тенниса и фитнеса

**Версия:** 1.0
**Дата:** 25 июня 2026
**Статус:** Реализовано (v1), готово к доработке

---

## 1. Обзор

### 1.1 Проблема

Игроку-любителю в настольный теннис нужно держать в одном месте три разных вида данных, которые сегодня разбросаны по разным источникам: технику и физоподготовку (видео из YouTube, советы тренеров), регулярные тренировки и таймеры (отдельные секундомер-приложения) и историю турниров с динамикой рейтинга (сайт федерации, который не даёт удобной автоматической аналитики). Нужен единый карманный инструмент, который объединяет тренировочный контент, таймер и учёт результатов — без необходимости в аккаунте, сервере или установке приложения.

### 1.2 Решение

Веб-приложение в виде одного HTML-файла («ТопСпин»), открывающееся в браузере телефона. Хранит все данные локально на устройстве и не требует подключения к интернету после первой загрузки (кроме просмотра встроенных видео).

### 1.3 Целевой пользователь

Один человек (владелец продукта) — игрок-любитель в настольный теннис, который:
- регулярно тренирует технику и физподготовку;
- участвует в любительских турнирах, зарегистрированных на RTTF.ru;
- хочет видеть прогресс по тренировкам и динамику рейтинга в одном месте.

### 1.4 Платформа и ограничения

| Параметр | Решение |
|---|---|
| Платформа | Браузер на телефоне (мобильный веб) |
| Установка | Не требуется; можно добавить на главный экран как PWA-ярлык |
| Хранение данных | `localStorage` браузера — данные привязаны к конкретному устройству/браузеру |
| Аккаунт/логин | Не нужен |
| Интернет | Не нужен, кроме загрузки видео-демонстраций (YouTube embed) |
| Интеграция с RTTF.ru | Только вручную — у сайта нет публичного API, и он блокирует автоматический доступ ботов (запрещено в `robots.txt`). Подробности в разделе 4.5 |

## 2. Цели и метрики успеха

| Цель | Как измеряется |
|---|---|
| Снизить трение при логировании тренировки/матча | Запись тренировки или матча занимает ≤ 30 секунд |
| Помочь закрепить технику упражнений | У каждого упражнения есть видео-демонстрация техники |
| Дать наглядность прогресса | График рейтинга и статистика тренировок видны без дополнительных действий на Главной |
| Удержать привычку тренироваться | Счётчик "дней подряд" виден на Главной экране |

## 3. Пользовательские истории

1. **Как игрок**, я хочу за одну тренировку открыть список упражнений по технике, посмотреть видео и отметить выполненное — чтобы не искать ролики по разным источникам.
2. **Как игрок**, я хочу запустить интервальный таймер с настраиваемыми раундами работы/отдыха — чтобы тренироваться по схеме без отдельного приложения-секундомера.
3. **Как игрок**, я хочу после турнира быстро занести результат каждого матча (соперник, счёт по партиям, рейтинг до/после) — чтобы не держать это в голове или в блокноте.
4. **Как игрок**, я хочу видеть график изменения своего рейтинга во времени — чтобы понимать, расту я или стою на месте.
5. **Как игрок**, я хочу видеть общую статистику (тренировки за неделю, минуты активности, % побед, дни подряд) на главном экране — чтобы быстро оценить форму без захода в разные разделы.
6. **Как игрок**, я хочу быстро перейти на свой профиль RTTF.ru — чтобы свериться с официальным рейтингом перед тем, как занести его в приложение.

## 4. Функциональные требования

### 4.1 Главный экран (Home)
- Блок быстрых действий: начать тренировку по теннису, начать фитнес-тренировку, открыть таймер, записать матч.
- Сводка за неделю: количество тренировок, суммарные минуты активности, количество сыгранных матчей, % побед за всё время.
- Карточка текущего рейтинга RTTF с прямой ссылкой на сайт.
- Лента последней активности (последние тренировки и матчи).
- Счётчик "дней подряд" в верхней панели.

### 4.2 Упражнения по теннису
- Список упражнений с фильтрами по категориям (Техника, Подача, Передвижение, Серии).
- Каждая карточка упражнения при раскрытии показывает: встроенное видео техники выполнения, текстовое описание, чипы с ключевыми фокус-точками (например, "Вращение корпуса", "Контакт в верхней точке").
- Кнопка "Отметить как выполнено" — пишет запись в журнал тренировок (тип `tennis`).

### 4.3 Фитнес-упражнения
- Та же структура карточек, что и у теннисных упражнений, с собственным набором: общеразвивающие упражнения, отобранные по принципу прямой пользы для игры (взрывная сила ног, боковая реакция, кор, плечевой пояс).
- Фильтры по категориям (Кардио, Взрывная сила, Кор, Ноги, Плечи).

### 4.4 Таймер тренировок
- Кольцевой визуальный таймер с фазами: разминка → работа → отдых → следующий раунд → завершение.
- Настраиваемые параметры: длительность работы, отдыха, разминки, количество раундов.
- Готовые шаблоны: «Теннис-серии», «Табата», «Силовая», «Выносливость».
- Звуковой сигнал на переходе между фазами.
- По завершении — автоматическая запись тренировки в журнал с подсчётом суммарной длительности.

### 4.5 Матчи и рейтинг
- Форма ручного занесения матча: соперник, дата, турнир/площадка, результат (победа/поражение), счёт по партиям (произвольное количество партий), рейтинг до и после матча.
- При открытии формы поле "рейтинг до" автоматически подставляется из последнего занесённого "рейтинга после" — чтобы не искать число повторно.
- График динамики рейтинга по всем матчам, где указан рейтинг после игры.
- Список истории матчей с возможностью удаления записи.
- Информационная карточка с прямой ссылкой на `rttf.ru` и пояснением, что синхронизация только ручная.

> **Решение по интеграции с RTTF.ru:** на этапе планирования была изучена возможность автоматического получения рейтинга и результатов турниров с сайта `rttf.ru`. Сайт не предоставляет публичного API и явно запрещает автоматический доступ ботов через `robots.txt`. Реализация автоматического парсинга вопреки этому запрету не рассматривалась. Вместо этого приложение даёт быстрый ручной флоу занесения результата (с автоподстановкой стартового рейтинга) и прямую ссылку на сайт для проверки актуальных цифр. Дополнительный фактор, снижающий цену вопроса: официальный рейтинг на сайте обновляется не чаще одного раза в неделю.

### 4.6 Прогресс
- Карточки: всего тренировок, всего минут, дней подряд.
- Столбчатая диаграмма тренировок за последние 7 дней.
- Баланс типов тренировок (теннис vs фитнес) в виде шкалы.
- Полный журнал тренировок с возможностью удаления записи.
- Кнопка полного сброса данных приложения (с подтверждением).

### 4.7 Навигация
- Нижняя таб-панель с пятью разделами: Главная, Теннис, Фитнес, Таймер, Матчи, Прогресс.

## 5. Нефункциональные требования

| Требование | Реализация |
|---|---|
| Автономность | Приложение — один self-contained HTML-файл; работает без сервера и сборки |
| Сохранность данных между визитами | `localStorage`, ключ `topspin_app_data_v1`, JSON со списками тренировок и матчей |
| Отказоустойчивость хранилища | Операции записи/чтения обёрнуты в try/catch с уведомлением пользователю при сбое |
| Адаптивность | Мобильный-first дизайн, безопасные зоны (`env(safe-area-inset-bottom)`) под нижнюю панель iOS |
| Доступность | Видимый фокус для клавиатурной навигации, уважение `prefers-reduced-motion` |
| Приватность | Все данные остаются на устройстве пользователя; ничего не отправляется на сервер |

## 6. Дизайн

Визуальный язык отталкивается от предметной области: тёмно-зелёный — цвет столешницы для пинг-понга, разметка стола — пунктирная линия-разделитель в шапке, акцентный цвет — оранжево-красный (цвет тренировочного мяча). Кольцевой таймер визуально отсылает к траектории мяча. Подробная токен-система (цвета, типографика, layout) зафиксирована в коде приложения (`<style>`-блок `index.html`).
